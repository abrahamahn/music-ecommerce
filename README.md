# REACT NATIVE TEMPLATE

This React Native template was made to quickly begin project development from scratch in an optimal environment:

This following template includes:
1. All of the latest updates of React Native, Node modules.
2. Latest tech stacks installed.
3. NPM package module dependencies checked.
4. Optimized folder structure for app development efficiency.

Folder structure styling was implemented with a reference of artical from the article 'Best Folder Structure for React Native Project'
written by Muskan Jain (uri: <https://learn.habilelabs.io/best-folder-structure-for-react-native-project-a46405bdba7>)
